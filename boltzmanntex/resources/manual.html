BoltzmannTex User Manual

BoltzmannTex is alpha quality software. While steps have been taken in the subprograms to create accurate results, this software, in its current state should not be depended on for actual real world use.

Please use this software at your own risk and not as your primary method of computation.

Introduction

BoltzmannTex is a program for performing predefined transformations and visualisations on acquired data using simple Graphical User Intefaces. This software is mainly meant for well defined techniques that can be rigidly written as a program, and do not require regular testing, editing or substantial knowledge of programming for usage) for usage (such as in the case of python scripts).

BoltzmannTex aims to be modular enough to be easily extendable for more sub-programs.

Getting Started

BoltzmannTex, when first launched, displays a welcome screen with a list of options. these options are called Subprograms and allow you to perform a certain computation.

Using Interactive Graphs Within the Program

The interactive graphs presented to the user by the program can be zoomed into, panned and scaled based on individual axes using the mouse. The steps to perform each of these operations are as follows:

  • To pan or move the graph: Click and drag the left mouse button.
  • To zoom in and out of the graph: Scroll up on down of the graph while the mouse is above the graph.
  • To scale an individual axis: Put the mouse around the label of one of the axes, and scroll up and down.

Available Subprograms

BoltzmannTex currently has 2 functional Subprograms available:

  1. Optical Emmision Spectroscopy Temperature Calculation
  2. Simple Graph Plotting

Optical Emmision Spectroscopy Temperature Calculation

This subprogram allows the user to perform temperature calculation of thermal plasma using optical emission spectroscopy based on the spectral signatures of the material being examined.

Material Data Input Screen

This screen allows the user to input the material data of the spectrum that is being analysed. Four Parameters are required for each emmision line in the spectrum: The Wavelength, the Transition probability, the Upper Level Energy and the Degeneracy, and are input into the table area available on the screen. The number of rows in the table is the number of emmision lines that will be considered by the program. When this list is populated by the user, the program is allowed to move to the next step.

Providing Material Data Input as File

Material data input can also be provided by importing a delimiter-separated table of values into the program by clicking the "Import" button visible. To prepare this table, one can simply use a text editor to enter each value, separate them by space for each column, and by a new line ("enter" key) for each row.

A second method would be to use a word processor or a spreadsheet program such as Microsoft Office or Libreoffice. One can prepare a graphical table of each value, then copy the entire table, and then paste it into a text editor. The file then can be opened by the program.

Spectral Data Input Screen

This screen allows the user to input the data of the spectral readings of the observation, as well as input the defined emission line ranges (or peaks) within the readings. When the correct number of peak ranges are supplied to the program, the program is allowed to proceed to the next step.

Peak Area Computation Screen

This screen shows all the areas of the individual peaks defined.

Regression Analysis Screen

This screen shows each of the observation points taken into consideration in the regression analysis phase or the linear fitting phase of the calculation. The user is shown a graph with all the points as well as the fitted line. On the right the numerical values of each point are shown in the order of the peaks. The user is also given an option to discard any of the peaks from the final calculation of the temperature by clicking a checkbox next to each entry.

The result is immediately updated on clicking one of the entries.

Result Screen

This screen shows the final resultant temperature, the correlation coefficient of the linear regression data, and the coefficient of determination (R2) value of the fitting.

A button for generating a report from the available data and another button is available here for saving the images of the plots from the program.
